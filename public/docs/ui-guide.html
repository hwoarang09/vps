Fab Group System

Fab들을 수동으로 그룹핑하여 그룹별 평균 통계를 비교하는 기능.

3 View Modes

  • Individual — 기존 Fab별 리스트 (기본)
  • Group — 그룹별 집계 (평균 throughput, speed 등)
  • Editor (⚙) — 그룹 생성/편집 (TanStack Table)

Group Editor

TanStack Table 기반. 데이터 드리븐 — 파라미터 추가 시 PARAM_COLUMNS 배열에 한 줄만 추가.

사용 흐름

  1. ⚙ 버튼 클릭 → Editor 모드 진입
  2. 컬럼 헤더 클릭 → 해당 파라미터로 정렬 (같은 값끼리 모임)
  3. 행 체크박스로 fab 선택
  4. 그룹 이름 입력 → "그룹 생성" 클릭
  5. Group 모드로 전환 → 그룹별 평균 통계 비교

Auto-hide Columns

Fab간 값이 동일한 컬럼은 자동 숨김. 예: BPR c=0 그룹 안에서는 γ 컬럼 안 보임.
